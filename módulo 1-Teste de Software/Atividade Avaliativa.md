# 📄Atividade avaliativa

### Produzir um texto curto (até 1 página) que responda:
    
    -   O que é teste de software (com suas palavras):
    -   Por que testar é importante para o negócio?
    -   O que caracteriza um software com qualidade?
    -   Qual o papel do QA em um time moderno?


*Na minha opinião, o teste de software é o processo de verificar se um sistema está funcionando corretamente e cumprir o que foi planejado.*
 *O principal objetivo é encontrar erros antes que o software seja entregue para os usuários.*  
 *Testar é importante para o negócio porque evita problemas, reduz custos com correções e aumenta a confiança dos clientes no sistema.*  
 *Quanto mais cedo um erro é encontrado, fica mais fácil e barato para corrigir. Um software com qualidade é aquele que funciona bem, atende ao que foi solicitado, é seguro, fácil de usar e apresenta poucos erros; assim, o usuário consegue utilizar sem dificuldades.* 
 *O papel do QA é ajudar a garantir essa qualidade. Além de realizar testes, o QA trabalha junto com a equipe para identificar problemas, sugerir melhorias e ajudar a entregar um software mais eficiente e  confiável para os usuários.*
