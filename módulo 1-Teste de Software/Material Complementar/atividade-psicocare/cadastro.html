Regras aplicadas nesta atividade

Validações do cadastro

  • Todos os campos são obrigatórios.
  • A senha deve possuir no mínimo 8 caracteres.
  • Usuário e e-mail não podem ser duplicados.
  • As informações são salvas apenas no armazenamento local do navegador.

Segurança e autenticação

  • Credenciais incorretas geram uma mensagem genérica.
  • Após 3 erros consecutivos, o acesso fica bloqueado por 15 minutos.
  • Tentativas de login são registradas para auditoria.
  • Áreas restritas exigem uma sessão autenticada.

Premissas da versão

  • Sem login por redes sociais.
  • Sem recuperação de senha.
  • Base pré-cadastrada para testes.
  • Ambiente de testes separado pela identificação dos dados locais.

Compatibilidade

  • Layout responsivo para computador e celular.
  • Uso de recursos suportados pelos principais navegadores modernos.
  • Processamento local, sem chamadas externas.
